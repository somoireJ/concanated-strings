# concanated-strings
javascript code on strings
Create a javascript program that takes in a string of words separated by spaces and performs the following actions:
 Converts the string to an array of words using the .split() method.
 Sorts the array of words in alphabetical order using the .sort() method.
 Reverses the order of the array using the .reverse() method.
 Extracts every second word from the reversed array using the .slice() method.
 Concatenates the extracted words together using the "+" operator.
 Displays the final concatenated string in all uppercase letters using the .toUpperCase() method.
